How to play
👆
Tap an arrow to send it escaping off the grid — but only if its entire path to the edge is clear of other arrows.
🚫
Blocked arrow? If arrows are in the way, tapping costs a life. You have limited lives per level.
💡
Use hints to highlight a safe arrow when you're stuck. Plan the order — clearing one arrow opens paths for others.
🏆
Clear all arrows from the grid to complete the level and unlock the next one.
